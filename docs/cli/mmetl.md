## mmetl

ETL tool to transform the export files from different providers to be compatible with Mattermost.

### Options

```
  -h, --help   help for mmetl
```

### SEE ALSO

* [mmetl check](mmetl_check.md)	 - Checks the integrity of export files.
* [mmetl grid-transform](mmetl_grid-transform.md)	 - Transforms a slack enterprise grid into multiple workspace export files.
* [mmetl sync-import-users](mmetl_sync-import-users.md)	 - Checks if any users in the export file already exist in the Mattermost instance, and makes sure both username and email are the same in both cases.
* [mmetl transform](mmetl_transform.md)	 - Transforms export files into Mattermost import files
* [mmetl version](mmetl_version.md)	 - Prints the version of mmetl.
